📝 ToDoApp

A simple To-Do list application built with Java Swing.

🚀 Features

Add tasks

Mark tasks as completed

Remove tasks

🛠️ Requirements

Java JDK 17 or later

A Java IDE (e.g., IntelliJ, Eclipse) or terminal

💻 Running the Project

Compile the program:

javac src/com/todo/Main.java -d out

Run the program:

java -cp out com.todo.Main
